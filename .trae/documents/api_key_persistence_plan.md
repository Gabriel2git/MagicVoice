## 技术实现细节

1. **文件保存**:
   - 确保目录存在，不存在则创建
   - 确保有写入权限
   - 正确处理异常情况

2. **文件读取**:
   - 检查文件是否存在
   - 正确解析 JSON 格式
   - 处理文件格式错误的情况

3. **用户界面**:
   - 提供清晰的错误提示
   - 确保输入框能够正确接收用户输入
   - 提供明确的操作反馈

4. **测试**:
   - 测试各种边界情况
   - 确保功能在不同环境下都能正常工作

## 预期结果

- 用户只需要输入一次 API Key

- 设置窗口能够正确处理用户输入
- 程序在各种情况下都能稳定运行

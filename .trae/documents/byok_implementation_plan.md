### [ ] 1. 引入 config.json 配置文件
- **优先级**: P0
- **依赖**: 无
- **描述**:

- **成功标准**:

### [ ] 2. 添加首次运行的 GUI 引导
- **优先级**: P0
- **依赖**: 任务 1

## 实施顺序

1. **任务 1**: 引入 config.json 配置文件
2. **任务 2**: 添加首次运行的 GUI 引导
3. **任务 3**: 更新托盘菜单，添加设置选项
4. **任务 4**: 确保线程安全
5. **任务 5**: 测试边缘情况

## 技术实现细节

1. **配置文件管理**:
   - 使用 JSON 格式存储配置
   - 优先从 config.json 读取 API Key
   - 保持向后兼容，仍支持其他方式

2. **GUI 实现**:
   - 使用 tkinter 创建设置窗口
   - 窗口包含：输入框、保存按钮、说明文本
   - 窗口居中显示，小巧美观

3. **线程管理**:
   - 主线程运行 tkinter mainloop()
   - 后台线程处理键盘监听和音频处理
   - 使用线程安全的方式共享数据

4. **托盘菜单**:
   - 在现有菜单中添加设置选项
   - 点击设置选项时弹出设置窗口

## 预期结果

- 程序启动时会检查 config.json 文件
- 如果没有 API Key，弹出设置窗口引导用户输入
- 用户输入 API Key 后，程序将其保存并继续运行
- 托盘菜单中包含设置选项，允许用户随时修改 API Key
- 程序在各种情况下都能稳定运行
